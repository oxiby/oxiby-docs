# Oxiby documentation

This repository contains the documentation for **Oxiby**, an experimental programming language.

The documentation contains a a comprehensive, tutorial-style language guide.
Standard library and language reference documentation is planned.

## Read the documentation

You can read the documentation on the web at [the Oxiby website](https://www.oxiby.com/).

## Examples

Example Oxiby programs used in the language guide can be found grouped by chapter in the `examples` directory.

## Development

The documentation is written in Markdown using [mdBook](https://github.com/rust-lang/mdBook).
